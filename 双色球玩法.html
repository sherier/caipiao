普通单式提供自选及机选两种方式。

1、普通单式投注：红色球选择6个号码，蓝色球选择1个号码为一注。

2、复式：

红色球区号码个数（胆码 + 拖码）≥ 7。蓝色球区号码个数 ≥ 2

 （一）红色球号码复式：

红色球区号码个数 ≥ 7，蓝色球号码个数 = 1；

（二）蓝色球号码复式：

红色球区号码个数 = 6，蓝色球号码个数 ≥ 2；

（三）全复式：

红色球区号码个数 ≥ 7，蓝色球号码个数 ≥ 2。

3、胆拖

红色球区分为胆码及拖码，胆码为认为一定会中的号，拖码为可能会中的号。

红色球区号码个数（胆码（必须选择1-5个） + 拖码）≥ 7。蓝色球区号码个数 ≥ 1


胆拖玩法界面特殊处理：

胆码及拖码不能相似，一个号码只能被选中一次，如果在胆码中选择了此号码，再点击拖码的此号码，则自动把此号码改为拖码，反之同理。


开发注意：

1、复式投注注数计算方法：

设N =红色球区个数 ，M = 蓝色球区所选个数，Z = 胆拖投注注数。

则： Z = M*N*(N-1)(N-2)(N-3)(N-4)(N-5) / 6!


2、胆拖投注注数计算方法：

设 N =红色球区胆码个数（1 ≤ N ≤ 5），M = 红色球区拖码个数（6－N ≤ M ≤20），W = 蓝色球区所选个数，Z = 胆拖投注注数。

则： Z = combin(M,6-N)×combin(W,1)

          = {M! / [(6-N)!*(M- (6-N) )!] }*{W! / (W-1)!}。
